# Shared Credential Storage

This folder contains secret information about administration, infrastructure and maybe more.

## Set Up

Do one of the following:

1. `(cd && git clone https://github.com/uichcc/.password-store)`
2. `pass init <YOUR-KEY-FP> && pass git init && pass git clone https://github.com/uichcc/.password-store hcc`

The first way makes this password store _the one_ for the current user. The second way creates a _profile_ under **your own** password store.

## Public Keys

Everyone who can decrypt this credential storage (`.gpg-id`) must place a copy of the public key under folder [`public_keys`](./public_keys) and sign it.

## Extension

To calculate TOTP, install [pass-otp](https://github.com/tadfisher/pass-otp) on your system, which enables the `pass otp <account>` subcommand.

# 共享凭据仓库

本目录包含管理、基础设施以及别的机密信息。

## 创建

以下方式任选其一：

1. `(cd && git clone https://github.com/uichcc/.password-store)`
2. `pass init <YOUR-KEY-FP> && pass git init && pass git clone https://github.com/uichcc/.password-store hcc`

第一种方式将本密码仓库设置为当前用户的密码仓库。第二种方式在**你自己**的密码仓库里创建一个子仓库。

## 公钥

所有可以解密本凭据仓库的人（`.gpg-id`）必须在 [`public_keys`](./public_keys) 目录下放一份公钥并对它签名。

## 扩展

要计算 TOTP 则需在你的系统上安装 [pass-otp](https://github.com/tadfisher/pass-otp)。这让你可以使用 `pass otp <账户>` 这个子命令。
